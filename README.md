# AlexNet-bird-classifier-lowres
AlexNet모델을 활용한 저해상도인 조류 이미지 분류 프로젝트

## 🖥️ 프로젝트 소개
데이콘의 '저해상도 조류 이미지 분류 AI 경진대회'의 데이터를 활용
저해상도(64×64)이미지를 AlexNet에 학습시키기 위한 사이즈(224×224)로 만들기 위해 전처리를 진행
